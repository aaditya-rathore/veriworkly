## Philosophy

VeriWorkly is built on the principle that professional data should respect user privacy by default. The platform is:

- **Local-First**: Data remains in the user's browser until explicit synchronization is initiated.
- **Open-Source**: Transparent code distributed under the MIT license.
- **ATS-Optimized**: High-fidelity document exports designed for modern recruitment systems.
